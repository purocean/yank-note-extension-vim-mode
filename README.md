# Vim 模式插件

Vim 插件, 通过加载 monaco 的插件 - [monaco-vim](https://github.com/brijeshb42/monaco-vim) 实现.

> 目前仅支持最基础的光标移动指令和搜索指令

## 如何使用
- 启用/关闭 Vim 编辑模式: [状态栏]工具-Vim模式, 或者快捷键 `cmd/ctrl + alt + v`
